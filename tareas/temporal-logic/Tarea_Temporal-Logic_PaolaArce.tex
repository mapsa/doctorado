\section{Introduction}

El término lógica temporal (temporal logic) se utiliza para representar información de manera temporal en un contexto de lógica. La lógica temporal es un tipo de lógica modal introducida en 1960 por Arthur Prior bajo el nombre de Tense Logic \cite{sep-logic-temporal} que surge del interés en la relación entre el tiempo y la modalidad (atribuido al filósofo Diodorus Cronus).

La lógica temporal es usada para aclarar aspectos filosóficos acerca del tiempo, para definir expresiones semánticas temporales en lenguaje natural, para codificar conocimiento temporal en inteligencia artificial y como herramienta para manejar los aspectos temporales de la ejecución de programas computacionales.
